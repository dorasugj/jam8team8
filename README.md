#ゲーム概要
プレイ人数：１人
プレイ対象：ミニゲーム好き
ジャンル　：魔物探し
10秒以内に森の中に隠れた透明人間を
いかに早く探し当て、倒すかを競うミニゲーム


#プラットフォーム
画面を横にし、1920*1080p、16:9の画面比率なので、どの機種でも可能？


#ゲーム内容
ゲーム起動後、タイトル画面に移動。
難易度が３種類あるので、好きな難易度を選択。
選択後、ゲーム画面に移動。
10秒以内に画面をクリック、ないしタップしまくり、隠れている透明人間を全員見つけ出す。
時間内に全員見つけ出せればゲームクリア。
倒した敵の数とタイムでスコアが変わるので、より早く多く倒せばハイスコアを出す事が出来る。
